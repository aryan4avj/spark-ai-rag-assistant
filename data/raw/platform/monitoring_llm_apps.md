# Monitoring LLM Applications

Monitoring LLM applications requires more than basic API uptime checks.

## What to monitor

Teams should monitor latency, token usage, retrieval quality, answer quality, fallback rates, and failure patterns.

## Why traces matter

Tracing model calls, retrieval steps, and tool calls helps debug incorrect answers and cost spikes.
