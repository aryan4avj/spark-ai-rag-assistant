# CI CD Basics

Continuous Integration and Continuous Deployment help teams deliver software changes safely and repeatedly.

## Continuous Integration

Continuous Integration usually includes linting, automated tests, and build validation on pull requests.

## Continuous Deployment

Continuous Deployment or release automation packages the application, pushes artifacts, and deploys to target environments after checks pass.
