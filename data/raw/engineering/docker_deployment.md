# Docker Deployment Guide

Docker packages an application and its dependencies into a portable container image.

## Why Docker matters

Docker makes local development, testing, and deployment more consistent across environments.

## Deployment basics

A service image is built from a Dockerfile, then run with environment variables, ports, and volume settings as needed.
