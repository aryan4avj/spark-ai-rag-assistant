# Spark AI RAG Assistant

A Spark AI style internal knowledge assistant built with:
- FastAPI
- Mistral
- Qdrant
- Docker
- GitHub Actions
- Azure deployment

## Goal
Build a production-style RAG assistant that ingests Confluence-like documents, stores embeddings in a vector database, retrieves relevant context, and answers grounded questions with citations.

## Planned features
- Document ingestion
- Chunking and metadata preservation
- Embeddings with Mistral
- Vector search with Qdrant
- RAG answer generation
- FastAPI backend
- Simple frontend
- Testing and evaluation
- Docker deployment
- Azure deployment
- CI/CD pipeline
